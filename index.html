Learning Plan Assistant

A free, open-source tool to support staff who are under pressure with NCCD compliance. Create learning plan notes and generate NCCD-aligned adjustment comments using non-deficit language. All data stays in your browser session.

Open Tool

Why I built this

  • adjustment notes Generate past-tense comments for NCCD documentation with structured prompts for engagement and outcomes.
  • learning plan Find barriers quickly, use non-deficit descriptions, and generate linked adjustments with frequency, duration and responsible staff fields.
  • non-deficit language Barriers focus on curriculum, environment and teaching factors rather than student deficits.
  • free & open source Designed to be shared and improved by educators. No logins, no cloud storage, no data transmission.
